## Visualization Types

### 1. Payoff Matrices

**Purpose**: Visualize strategic form games  
**Example**:
```
            Player 2
            C       D
Player 1  C (3,3)  (0,5)
          D (5,0)  (1,1)
```

**Files**: `game_theory/payoff_matrices/*.png`

### 2. Game Trees

**Purpose**: Represent sequential games  
**Features**:
- Decision nodes
- Chance nodes
- Terminal payoffs
- Information sets

**Files**: `game_theory/game_trees/*.png`

### 3. Nash Equilibrium Plots

**Purpose**: Visualize mixed strategy equilibria  
**Features**:
- Probability simplexes
- Best response functions
- Equilibrium points
- Indifference curves

**Files**: `game_theory/equilibria/*.png`

### 4. Auction Performance Charts

**Purpose**: Compare auction mechanisms  
**Metrics**:
- Revenue by auction type
- Efficiency rates
- Winner's curse magnitude
- Bidder payoffs

**Files**: `auctions/*/*.png`

### 5. Voting Rule Comparisons

**Purpose**: Evaluate voting mechanisms  
**Visualizations**:
- Winner by voting rule
- Condorcet cycles
- Strategic manipulation examples
- Social welfare comparisons

**Files**: `voting/*/*.png`

### 6. Experimental Results

**Purpose**: Present behavioral findings  
**Charts**:
- Strategy frequency over time
- Payoff distributions
- Human vs AI comparison
- Convergence analysis

**Files**: `experiments/*/*.png`

## Generation Scripts

All visualizations are generated by scripts in the [`computational_scientist/`](../computational_scientist/) folder:

```python
# Example: Generate payoff matrix visualization
from computational_scientist.scripts.visualizer import PayoffMatrixPlotter
import numpy as np

payoff_A = np.array([[3, 0], [5, 1]])
payoff_B = np.array([[3, 5], [0, 1]])

plotter = PayoffMatrixPlotter()
plotter.plot_bimatrix(payoff_A, payoff_B, save_path="visualizations/game_theory/payoff_matrices/prisoners_dilemma.png")
```

```python
# Example: Generate equilibrium plot
from computational_scientist.scripts.visualizer import EquilibriumPlotter

plotter = EquilibriumPlotter()
plotter.plot_mixed_strategy_equilibrium(
    payoff_A, payoff_B,
    save_path="visualizations/game_theory/equilibria/mixed_nash.png"
)
```

## Visualization Standards

### Format
- **PNG**: High-resolution raster images (300 DPI)
- **PDF**: Vector graphics for publications
- **SVG**: Scalable vector graphics for web

### Style Guidelines
- **Color Scheme**: Consistent color palette across all figures
- **Fonts**: Clear, readable fonts (Arial, Helvetica, or LaTeX)
- **Labels**: Descriptive axis labels and legends
- **Titles**: Informative titles for all figures
- **Size**: Standard sizes for consistency

### Naming Convention
```
<category>_<description>_<version>.png

Examples:
- game_theory_prisoners_dilemma_v1.png
- auction_revenue_comparison_v2.png
- voting_condorcet_paradox_v1.png
- experiment_human_vs_ai_v3.png
```
